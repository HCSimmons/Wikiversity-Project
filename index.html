Using Wikiversity in the Classroom

This is the base repo for the WRIT 350 technical documentation project.

Our group created a lesson plan for a middle school teacher to use in the classroom. This lesson plan described an activity where students used Wikiversity, among other Wiki medias, to complete a project about a particular culture.
